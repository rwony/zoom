# Zoom

Zoom Clone using NodeJS, WebRTC and Websockets.


### Features
1. Video call
2. User screen share
3. Mute & Unmute
4. Turn Camera On & Turn Camera Off

### Languages
* NodeJS
* Javascript(ES6)
* SCSS
* Pug
